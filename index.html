Research Chatbot

Developed an AI-powered research assistant that enables users to ask questions about PDF documents and receive contextually relevant answers. This project implements Retrieval-Augmented Generation (RAG) using a combination of FAISS for vector search and TinyLlama-1.1B-Chat for natural language generation. The chatbot processes PDFs by chunking text into overlapping segments, embedding them with all-MiniLM-L6-v2, and indexing them using FAISS for efficient similarity search. The retrieved chunks serve as context for TinyLlama-1.1B, generating high-quality responses.
